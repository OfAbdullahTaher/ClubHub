# ClubHub
### College Clubs Engagement Platform

King Faisal University – College of Computer Science & Information Technology  
Graduation Project – Second Semester 2025–2026  
Supervisor: Dr. Raid Alzubi  

## Overview

ClubHub is a centralized digital platform designed to enhance student engagement in college clubs at King Faisal University.

The platform addresses the absence of a unified and structured mechanism for managing extracurricular participation. Currently, club information is scattered across informal channels such as social media, resulting in limited visibility, inconsistent access, and fragmented participation procedures.

ClubHub introduces a structured digital governance framework that centralizes club information, standardizes membership processes, and enables accurate tracking and analysis of student engagement.

## Problem Statement

Many students face difficulty identifying available clubs, understanding their objectives, and accessing clear application procedures.

The current environment presents several challenges:
- Fragmented dissemination of club information
- Inconsistent membership procedures
- Manual or unverified attendance tracking
- Lack of measurable engagement indicators

These issues create a socio-organizational gap characterized by limited awareness, unequal accessibility, and absence of reliable participation data.

## Project Objectives

- Develop a centralized web-based platform for managing college clubs
- Provide structured club profiles to support informed student decisions
- Standardize membership application procedures
- Implement secure QR-based attendance verification
- Design a Student Participation KPI framework
- Establish a participation-based points and incentive system
- Provide role-based dashboards (Student, Club Leader, Administrator)
- Ensure scalable system architecture for future expansion

The initial implementation is limited to one college and excludes financial systems or external integrations.

## Intelligent Recommendation System (AI Component)

ClubHub integrates an AI-driven recommendation module designed to increase meaningful participation.

The system analyzes:
- Student profile information
- Declared interests
- Previous club memberships
- Event attendance history
- Platform browsing behavior (visited clubs and viewed events)

Based on these inputs, the AI engine generates personalized club and event recommendations tailored to each student.

### Implementation Phases

**Phase 1**
- Rule-based filtering and scoring model

**Future Phase**
- Machine learning recommendation model trained on engagement patterns

This component transforms the platform from a static directory into a dynamic, data-driven engagement system.

## System Roles

### Student
- Browse clubs and events
- Receive AI-based recommendations
- Apply for membership
- Register for events
- Track participation points

### Club Leader
- Manage club profile
- Create and manage events
- Approve membership requests
- Monitor attendance and engagement metrics

### Administrator
- Approve clubs
- Manage users
- Monitor system-wide KPIs
- Access institutional performance analytics
